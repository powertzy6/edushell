| Attribute | Detail |
|-----------|--------|
| **Age** | 14 |
| **Role** | Siswa SMP Kelas 8 |
| **Location** | Jawa Tengah, kota kecil |
| **Device** | Laptop bekas, Intel Celeron N3060, 4GB RAM, SSD 120GB |
| **Previous OS** | Windows 10 (lambat) |
| **Technical Level** | Rendah. Bisa buka browser, Microsoft Office dasar, main game ringan. |
| **Goals** | Belajar online, ngerjain tugas, nonton video pembelajaran |
| **Pain Points** | Windows lemot, sering crash, bingung dengan Linux |
| **Expectations** | Desktop harus "berfungsi begitu dinyalakan" |

### Adi's Workflow
1. Nyalakan laptop → Login → Muncul desktop bersih
2. Klik ikon browser → Buka Google Classroom
3. Download file tugas → Klik dua kali → Terbuka di LibreOffice
4. Selesai → Klik menu → Shutdown
5. Selesai. Tidak perlu terminal. Tidak perlu troubleshooting.

### What Adi Needs
- Browser dengan bookmark pendidikan
- Office suite terinstal dan terlihat
- Launcher yang mudah ditemukan
- Settings sederhana
- WiFi dan volume mudah diatur
