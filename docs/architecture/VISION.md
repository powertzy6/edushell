# EduShell Vision Document

## Project Title
**EduShell** — A modern desktop shell for Indonesian students.

## Vision Statement
Menjadi Desktop Shell Linux paling ramah untuk pelajar Indonesia — menghilangkan hambatan teknis, menghadirkan pengalaman komputasi modern yang setara dengan platform mainstream, dan membangun ekosistem open source education-first di atas fondasi Linux.

To become the most student-friendly Linux Desktop Shell for Indonesia — removing technical barriers, delivering a modern computing experience equal to mainstream platforms, and building an open-source education-first ecosystem on Linux foundations.

## Long-term Goal (5 Years)
Transformasi bertahap dari Desktop Shell berbasis Cinnamon menjadi **Desktop Environment mandiri** (EduDE) yang sepenuhnya independen, dengan Window Manager, compositor, dan application stack sendiri — tanpa mengorbankan stabilitas dan kompatibilitas.

## Key Differentiators
1. **Education-first**: Setiap elemen UI dirancang untuk mempercepat pembelajaran, bukan produktivitas power user.
2. **Indonesian-first**: Bahasa Indonesia sebagai first-class citizen, bukan sekadar terjemahan tempelan.
3. **Gentle learning curve**: Transisi dari Windows/macOS tanpa dokumentasi dan zero confusion.
4. **Resource-conscious**: Berjalan mulus di hardware kelas bawah (4GB RAM, Celeron).
5. **Progressive independence**: Dari shell atop Cinnamon menuju DE mandiri secara bertahap.

## Target Ecosystem
- Distribusi: Ubuntu LTS, Linux Mint, Debian Stable
- Display Server: Wayland (primer), X11 (kompatibilitas)
- Bahasa: Bahasa Indonesia, English
- Lisensi: GPL-3.0-or-later
