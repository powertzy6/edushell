# EduShell v1.0 — Branding Guidelines

## Logo
- Use the EduShell logo with the "E" icon and text
- Minimum clear space: 1x the height of the "E"

## Color Palette
- Primary: #3584e4 (Blue)
- Success: #35e435 (Green)
- Warning: #e4a035 (Orange)
- Danger: #e43535 (Red)
- Dark BG: #1a1a2e
- Light BG: #ffffff

## Typography
- Primary font: Noto Sans, sans-serif
- Code font: Noto Sans Mono, monospace
- Base size: 12px

## Usage
- Always use the official logo for downloads, documentation, and marketing
- Keep the "Educational Desktop Shell" tagline in prominent positions
- Refer to the project as "EduShell" (capital E, capital S)
